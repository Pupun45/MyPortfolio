Title: MyPortfolio - ASP.NET Core Personal Portfolio

Description:
This is a personal portfolio web application built using ASP.NET Core and Razor Pages. The project showcases my skills, work experience, projects, and provides interactive ways for visitors to contact me. The portfolio is fully responsive and includes modern UI features such as animations, scroll reveal effects, and an interactive resume preview.

Features:

Home, About, Skills, Resume, Contact Pages built with Razor Pages.

Resume Section: Downloadable PDF and live preview embedded in the site.

Contact Page: Send emails directly through Gmail or default mail client.

Responsive Design: Works seamlessly on desktop, tablet, and mobile.

Tailwind CSS: For modern, responsive UI styling.

Animations: Smooth scroll reveal and hover effects for interactivity.

Tech Stack:

Backend: ASP.NET Core 8 / Razor Pages

Frontend: HTML5, Tailwind CSS, JavaScript

Hosting Ready: Can be deployed on IIS, Azure, or other hosting platforms
